# CS-128H-Group-12-Project
This is proposed by Renyu Liu (renyul2)
Project name: Riichi Mahjong Assistant

Background: The scoring rules in Riichi Mahjong is complicated for players to calculate with their hands. So this project would help users to calculate the points when users have won a game. 

Input: Users will type in: 
1:the 14 tiles they used to win, indicating the winning tile; 
2: indicate which tiles are dora; 
3: indicate the prevalent wind and seat wind; 
4: check true / false: trumo/ron, concealed hand/open hand, Riichi/Ippatsu
Function: 
1: check and identify the type of yaku
2: calculate fu, han, to get the points
Output: the points users get from winning this game
